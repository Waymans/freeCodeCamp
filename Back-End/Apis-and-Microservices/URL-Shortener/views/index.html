URL Shortener Microservice

Project from freeCodeCamp.
  1. I can POST a URL to [project_url]/api/shorturl/new and I will receive a shortened URL in the JSON response. Example : {"original_url":"www.google.com", "short_url":a9o2}.
  2. If I pass an invalid URL that doesn't follow the valid http(s)://www.example.com(/more/routes) format, the JSON response will contain an error like {"error":"invalid URL"}.
  3. When I visit the shortened URL, it will redirect me to my original link.
Can use the submit form to shorten!

example: https://url-shortener1-project.glitch.me/api/shorturl/new/https://www.freecodecamp.com

shortened url: https://url-shortener1-project.glitch.me/api/shorturl/d8e0 will redirect to: https://www.freecodecamp.com
